# Sun West Professional Window Tinting Website

Static one-page website for Sun West Professional Window Tinting.

## Files

- `index.html` — main website page
- `styles.css` — layout, colors, typography, responsive design
- `script.js` — mobile menu and quote-message helper
- `assets/images/` — optimized project images used on the website
- `assets/favicon.svg` — simple SW favicon
- `netlify.toml` — Netlify security headers and image caching
- `robots.txt` — allows search engines to crawl the site

## Security / privacy choices included

- No third-party scripts
- No external fonts or CDNs
- No API keys or passwords
- No checkout or payment collection
- Quote form does not store or send data to a server; it only helps the visitor create a text message
- Security headers included for Netlify
- Visible license plate area in the main rear vehicle photo was blurred before export
- Business card and review screenshots are not included in the public site files, so the owner's last name is not displayed in the site assets

## Netlify deployment

Upload this ZIP to Netlify or connect it to a GitHub repo. Netlify will read `netlify.toml` automatically.

## Editing

Update text inside `index.html`.
Update colors and spacing inside `styles.css`.
Add photos to `assets/images/` and reference them in the Gallery section of `index.html`.
